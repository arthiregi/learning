# தமிழ் மாதங்கள் மற்றும் வேளாண் பயிரிடும் காலங்கள்
## (Tamil Months and Agricultural Seasons)

இந்த ஆவணம் தமிழ்நாட்டில் வேளாண்மைக்கான அடிப்படை அறிவை வழங்குகிறது. தமிழ் மாதங்கள், பருவங்கள் மற்றும் எந்த மாதத்தில் எந்தப் பயிரை எப்படி வளர்ப்பது என்பதற்கான வழிகாட்டியாக இது உள்ளது.

---

## 1. தமிழ் மாதங்கள் (Tamil Months)

தமிழ் நாட்காட்டி சூரிய மற்றும் சந்திர இயக்கங்களை அடிப்படையாகக் கொண்டது. ஒவ்வொரு மாதமும் 30 நாட்களைக் கொண்டது. இங்கு 12 தமிழ் மாதங்களும் அவற்றின் கிரெகோரியன் மாதங்களும் தரப்பட்டுள்ளன[citation:1][citation:2]:

|:---:|:---|:---|:---|
| 1 | சித்திரை | Chittirai | mid-April – mid-May |
| 2 | வைகாசி | Vaikāsi | mid-May – mid-June |
| 3 | ஆனி | Āni | mid-June – mid-July |
| 4 | ஆடி | Ādi | mid-July – mid-August |

| 8 | கார்த்திகை | Kārtikai | mid-November – mid-December |

| 11 | மாசி | Māsi | mid-February – mid-March |

## 2. தமிழ்நாட்டு நெல் பயிரிடும் பருவங்கள் (Rice Cultivation Seasons in Tamil Nadu)

தமிழ்நாட்டில் நெல் பயிரிட பல பருவங்கள் உள்ளன. ஒவ்வொரு பருவமும் குறிப்பிட்ட மாதங்களில் தொடங்கி வெவ்வேறு மாவட்டங்களில் பயிரிடப்படுகிறது[citation:4][citation:6][citation:10]:

| பருவப் பெயர் | விதைக்கும் மாதம் | கால அளவு (நாட்கள்) | பயிரிடும் மாவட்டங்கள் |
|:---|:---:|:---:|:---|
| **நவரை** (Navarai) | டிசம்பர் - ஜனவரி | < 120 | திருவள்ளூர், வேலூர், திருவண்ணாமலை, கடலூர், விழுப்புரம், திருச்சிராப்பள்ளி, பெரம்பலூர், கரூர், நாகப்பட்டினம், மதுரை, தேனி, சேலம், நாமக்கல், திண்டுக்கல், தர்மபுரி, கோயம்புத்தூர், ஈரோடு, புதுக்கோட்டை |
| **சோர்ணவரி** (Sornavari) | ஏப்ரல் - மே | < 120 | திருவள்ளூர், வேலூர், திருவண்ணாமலை, கடலூர், விழுப்புரம், நாமக்கல், தர்மபுரி |
| **கார்** (Kar) | மே - ஜூன் | < 120 | திருநெல்வேலி, கன்னியாகுமரி, தூத்துக்குடி, ஈரோடு, கோயம்புத்தூர், மதுரை, தேனி, திண்டுக்கல், சேலம், நாமக்கல், தர்மபுரி |
| **குறுவை** (Kuruvai) | ஜூன் - ஜூலை | < 120 | திருச்சிராப்பள்ளி, பெரம்பலூர், கரூர், தஞ்சாவூர், நாகப்பட்டினம், திருவாரூர், புதுக்கோட்டை, ஈரோடு |
| **ஆரம்ப சம்பா** (Early Samba) | ஜூலை - ஆகஸ்ட் | 130 - 135 | திருவள்ளூர், வேலூர், திருவண்ணாமலை, சேலம், நாமக்கல், கடலூர், விழுப்புரம், மதுரை, தேனி, இராமநாதபுரம், தர்மபுரி, கோயம்புத்தூர், ஈரோடு, புதுக்கோட்டை, நீலகிரி |
| **சம்பா** (Samba) | ஆகஸ்ட் | 130 - 135 மற்றும் >150 | அனைத்து மாவட்டங்களும் |
| **தளாடி / பிசாணம்** (Thaladi / Pishanam) | செப்டம்பர் - அக்டோபர் | 130 - 135 | திருவள்ளூர், மதுரை, தேனி, கோயம்புத்தூர், ஈரோடு, திண்டுக்கல், கன்னியாகுமரி, திருநெல்வேலி, தூத்துக்குடி |
| **பிற்பட்ட தளாடி** (Late Thaladi) | அக்டோபர் - நவம்பர் | 115 - 120 | தஞ்சாவூர், நாகப்பட்டினம், திருவாரூர், திருச்சிராப்பள்ளி, பெரம்பலூர், கரூர் |

## 4. மாதங்களின் விவசாய முக்கியத்துவம் (Agricultural Significance by Month)

| மாதம் | விவசாய நடவடிக்கைகள் |
|:---|:---|
| **சித்திரை** | புத்தாண்டு தொடக்கம். நிலத்தை உழத் தொடங்கும் காலம். விவசாயக் கடன்கள் மற்றும் ஒப்பந்தங்கள் புதுப்பிக்கப்படும்[citation:8]. |
| **வைகாசி** | மழை குறைவு, ஆறுகள் வறண்ட நிலை. சோர்ணவரி நெல் விதைப்பு தொடங்கும்[citation:4]. |
| **ஆனி** | நிலம் உழுதல், எருவிடுதல் மற்றும் விதைத்தல் தொடர்கிறது. |
| **ஆடி** | குறுவை மற்றும் கார் நெல் பயிரிடும் உச்ச காலம். இந்த மழை குறுவை பயிருக்கு முக்கியமானது[citation:4][citation:7]. |
| **ஆவணி** | சம்பா நெல் விதைக்கும் காலம். நிலத்தில் நீர் தேங்கி நிற்கும்[citation:4]. |
| **புரட்டாசி** | சம்பா மற்றும் தளாடி நெல் விதைப்பு[citation:4]. கேழ்வரகு புரட்டாசிப்பட்டம் (செப்-அக்) விதைக்கப்படும்[citation:11]. |
| **ஐப்பசி** | தளாடி / பிசாணம் நெல் விதைப்பு[citation:4]. |
| **கார்த்திகை** | குறுவை நெல் அறுவடை மற்றும் தளாடி நெல் வளர்ச்சி. |
| **மார்கழி** | நவரை நெல் விதைக்கும் காலம்[citation:4]. மார்கழிப்பட்டம் கேழ்வரகு[citation:11]. |
| **தை** | நவரை நெல் வளர்ச்சி. விவசாய வேலைகள் குறையும். |
| **மாசி** | நவரை நெல் அறுவடை. |
| **பங்குனி** | புதிய விவசாய ஆண்டிற்கு தயாராகும் காலம். |
