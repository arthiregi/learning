# ராசிபுரத்தில் காய்கறி சாகுபடி - அடிப்படை வழிகாட்டி
## (Fundamental Guide for Vegetable Cultivation in Rasipuram)

---

## 1. ராசிபுரத்திற்கான சாகுபடி அடிப்படைகள்

ராசிபுரத்தில் காய்கறி சாகுபடிக்கு பாதுகாக்கப்பட்ட சாகுபடி முறைகள் (Protected Cultivation) மிகவும் பயனுள்ளதாக உள்ளது. பாலிஹவுஸ், ஷேட் நெட், கிரீன்ஹவுஸ் போன்ற கட்டுப்படுத்தப்பட்ட சூழல் முறைகள் பயிர்களை தீவிர வெப்பம், பூச்சிகள், நோய்களிலிருந்து பாதுகாத்து, 5 மடங்கு வரை அதிக மகசூல் தருகின்றன[citation:1].

**முக்கிய பயிரிடும் கட்டமைப்புகள்:**
- **பாலிஹவுஸ்**: காய்கறிகள் மற்றும் பூக்களுக்கு ஏற்றது
- **ஷேட் நெட்**: அதிக சூரிய ஒளியிலிருந்து பாதுகாக்கும்
- **ஹைட்ரோபோனிக்ஸ்**: மண் இல்லாமல் சாகுபடி, 90% நீர் சேமிப்பு

---

## 2. காய்கறிகளின் தேர்வு - பருவம், மண், நீர் தேவைகள்

### 2.1 பருவத்திற்கு ஏற்ற பயிர்கள்

| பருவம் | மாதங்கள் | பயிரிடும் காய்கறிகள்[citation:3] |
|:---|:---|:---|
| **காரிஃப் (Kharif)** | ஜூன்-அக்டோபர் | சுரைக்காய் வகைகள், கத்திரிக்காய், மிளகாய், வெண்டைக்காய், தக்காளி, இஞ்சி, மஞ்சள், மொச்சை, கொத்தவரை, சேங்கிழங்கு, தாமரைக்கிழங்கு |
| **ராபி (Rabi)** | அக்டோபர்-பிப்ரவரி | முட்டைக்கோஸ், காலிஃப்ளவர், கேரட், பீட்ரூட், முள்ளங்கி, உருளைக்கிழங்கு, பட்டாணி, வெங்காயம், பூண்டு, கீரை வகைகள் |
| **கோடை** | பிப்ரவரி-மே | கத்திரிக்காய், தக்காளி, மிளகாய், வெண்டைக்காய், மொச்சை, சுரைக்காய் வகைகள் |

### 2.2 மண் pH அடிப்படையில் பயிர் தேர்வு

| மண் pH வரம்பு | பொருத்தமான பயிர்கள்[citation:3] |
|:---|:---|
| 5.5 - 7.0 | கத்திரிக்காய், மிளகாய் |
| 5.5 - 7.5 | தக்காளி, சுரைக்காய், பூசணி, வெள்ளரி |
| 6.0 - 7.0 | கீரை, அவரை |
| 6.0 - 7.5 | புஷ் பீன்ஸ், பீட்ரூட், முள்ளங்கி, பச்சைப் பட்டாணி |
| 6.0 - 8.0 | வெண்டைக்காய், சர்க்கரை வள்ளிக்கிழங்கு |

### 2.3 நீர் தேவை அடிப்படையில் பயிர் வகைப்பாடு

| நீர் தேவை | பயிர்கள்[citation:3] |
|:---|:---|
| **மிகக் குறைவு** | தர்பூசணி, பூசணி, அஷ்கார்டு |
| **குறைவு** | கொத்தவரை, பிரெஞ்ச் பீன்ஸ், மொச்சை, பட்டாணி, பீர்க்கங்காய், சுரைக்காய் |
| **மிதம்** | தக்காளி, மிளகாய், வெங்காயம், உருளைக்கிழங்கு, கேரட், வெள்ளரி |
| **அதிகம்** | இலைக் காய்கறிகள், முட்டைக்கோஸ், காலிஃப்ளவர், ப்ரோக்கோலி |

## 3. ஒவ்வொரு காய்கறிக்கும் தனிப்பட்ட பராமரிப்பு (Crop-Specific Care)

### 3.1 கத்திரிக்காய் (Brinjal / Eggplant)

| பராமரிப்பு | விவரம் |
|:---|:---|
| **நடவு காலம்** | காரிஃப் (ஜூன்-அக்), ராபி (அக்-பிப்) |
| **மண்** | நன்கு வடிகால் உள்ள களிமண், pH 5.5-7.0[citation:3] |
| **நீர்** | மிதமான நீர் தேவை |
| **பராமரிப்பு** | வாராந்திர களை எடுத்தல், மண் கூட்டுதல், 15 நாட்களுக்கு ஒருமுறை உரமிடுதல் |
| **அறுவடை** | நடவு செய்து 60-70 நாட்களில் |
| **உதவும் பூச்சிகள்** | தேனீக்கள் மகரந்த சேர்க்கைக்கு, லேடிபட் பூச்சிகள் (Beetles) நுண்பூச்சிகளை கட்டுப்படுத்தும் |

### 3.2 தக்காளி (Tomato)

| பராமரிப்பு | விவரம் |
|:---|:---|
| **நடவு காலம்** | காரிஃப் மற்றும் ராபி, pH 5.5-7.5[citation:3] |
| **மண்** | நன்கு வடிகால் உள்ள மணல் களிமண் |
| **நீர்** | மிதமான நீர் தேவை, துளிச் சொட்டு நீர் பாசனம் சிறந்தது[citation:1] |
| **பராமரிப்பு** | கிளைகளை கத்தரித்தல், கொடிகளைக் கட்டுதல் |
| **அறுவடை** | 70-80 நாட்களில் |
| **இடைப்பயிர்** | தக்காளி + துளசி - புழுப்பூச்சிகளை கட்டுப்படுத்தும்[citation:3] |
| **உதவும் பூச்சிகள்** | தேனீக்கள், லேடிபட் பூச்சிகள், திருப்பி (Praying Mantis) |

### 3.3 உருளைக்கிழங்கு (Potato)

| பராமரிப்பு | விவரம் |
|:---|:---|
| **நடவு காலம்** | ராபி (அக்-பிப்)[citation:2] |
| **விதை விகிதம்** | 300-400 கிராம்/ச.மீ[citation:2] |
| **இடைவெளி** | 55 செ.மீ வரிசை, 20 செ.மீ செடி[citation:2] |
| **உரம்** | நன்கு மக்கிய தொழு உரம் 4-5 கிலோ/ச.மீ[citation:2] |
| **உரம் (துணை)** | அம்மோனியம் சல்ஃபேட் 60 கிராம், சூப்பர் பாஸ்பேட் 37.5-40 கிராம்[citation:2] |
| **மண் கூட்டுதல்** | 25 செ.மீ உயரத்தில் 15 செ.மீ உயரத்தில் மண் கூட்டவும்[citation:2] |
| **அறுவடை** | இலைகள் மஞ்சளாகும்போது, 60-130 நாட்களில்[citation:2] |
| **வகைகள் (ஆரம்ப)** | 'Kufri Chandramukhi', 'K. Jyoti', 'K. Moti', 'K. Lovkar'[citation:2] |
| **வகைகள் (முதன்மை)** | 'Kufri Chamtkar', 'K. Sheetman', 'K. Sindhuri', 'K. Jeevan', 'K. Naveen'[citation:2] |
| **கவனம்** | உருளைக்கிழங்கு வெடிப்பு (Blight) - தாமிர அயோடைடு தெளிக்கவும்[citation:2] |

### 3.4 கீரை வகைகள் (Leafy Vegetables - Amaranth, Spinach, Methi)

| பராமரிப்பு | விவரம் |
|:---|:---|
| **நடவு காலம்** | ஆண்டு முழுவதும் |
| **மண்** | pH 6.0-7.0[citation:3] |
| **நீர்** | அதிக நீர் தேவை |
| **ஒளி** | குறைந்த ஒளி தேவைப்படுபவை[citation:3] |
| **அறுவடை** | 20-30 நாட்களில் முதல் அறுவடை |
| **உரம்** | நைட்ரஜன் உரம் அதிகம் தேவை |
| **உதவும் பூச்சிகள்** | தேனீக்கள், லேடிபட் பூச்சிகள் |

### 3.5 மிளகாய் / குடைமிளகாய் (Chilli / Capsicum)

| பராமரிப்பு | விவரம் |

| **மண்** | pH 5.5-7.0[citation:3], நன்கு வடிகால் |
| **நீர்** | மிதமான நீர் தேவை[citation:3] |
| **வேரின் ஆழம்** | 90-120 செ.மீ ஆழமான வேர்கள்[citation:3] |

### 3.6 அவரை வகைகள் (Beans - French Bean, Cowpea, Cluster Bean)

| பராமரிப்பு | விவரம் |
|:---|:---|
| **நடவு காலம்** | காரிஃப், ராபி[citation:3] |
| **மண்** | pH 6.0-7.5[citation:3] |

| **உதவும் பூச்சிகள்** | தேனீக்கள் மகரந்த சேர்க்கைக்கு, டிரிக்கோகிரம்மா (Trichogramma) முட்டை கிருமியை கட்டுப்படுத்தும் |

| **வேரின் ஆழம்** | 120-180 செ.மீ ஆழமான வேர்கள்[citation:3] |
| **உதவும் பூச்சிகள்** | தேனீக்கள், வண்டுகள், சிறிய குளவிகள் |

## 4. பயிர் சுழற்சி (Crop Rotation) - முக்கிய கூறு

**பயிர் சுழற்சி** என்பது ஒரே நிலத்தில் வெவ்வேறு பயிர்களை முறை மாற்றி பயிரிடும் முறையாகும். இது நோய்கள், பூச்சிகளை கட்டுப்படுத்தவும், மண்ணின் கரிமப் பொருள் மற்றும் ஊட்டச்சத்துக்களை மேம்படுத்தவும் உதவுகிறது[citation:3].

**குடும்ப அடிப்படையில் பயிர்கள்:**

| குடும்பம் | பயிர்கள் | குறிப்பு |
|:---|:---|:---|
| **குக்குர்பிடேசி** (Cucurbitaceae) | வெள்ளரி, பூசணி, அனைத்து சுரைக்காய்கள் | அதிக நீர் தேவை, ஆனால் நீர் தேங்கிய மண்ணை விரும்பாது[citation:3] |
| **சோலனேசி** (Solanaceae) | கத்திரிக்காய், தக்காளி, மிளகாய், உருளைக்கிழங்கு | [citation:3] |
| **ஃபபேசி** (Fabaceae) | அனைத்து அவரை, பட்டாணி வகைகள் | ஆழமான வேர்கள், வறட்சியை தாங்கும்[citation:3] |
| **மால்வேசி** (Malvaceae) | வெண்டைக்காய் | [citation:3] |
| **செனோபோடியேசி** (Chenopodiaceae) | பீட்ரூட், கீரை வகைகள் | [citation:3] |

**பயிர் சுழற்சி திட்டம் - 4 படிகள்:**

| படி | பயிர் வகை | உதாரணம் |
|:---|:---|:---|
| **1** | இலை / பூ பயிர்கள் | கீரை, முட்டைக்கோஸ், காலிஃப்ளவர்[citation:3] |
| **2** | வேர் பயிர்கள் | வெங்காயம், உருளைக்கிழங்கு, கேரட்[citation:3] |
| **3** | காய் / பழ பயிர்கள் | தக்காளி, கத்திரிக்காய், மிளகாய்[citation:3] |
| **4** | பருப்பு வகைகள் | பட்டாணி, அவரை[citation:3] |

**அதிக ஊட்டம் தேவைப்படும் பயிர்களுக்கு பின் குறைந்த ஊட்டம் தேவைப்படும் பயிர்கள்** பயிரிடவும்[citation:3].

## 5. இடைப்பயிர் முறை (Intercropping)

இடைப்பயிர் முறையானது நிலத்தின் உற்பத்தித்திறனை அதிகரிக்கவும், பூச்சிகளை கட்டுப்படுத்தவும் உதவுகிறது[citation:9].

### 5.1 பூச்சி கட்டுப்பாட்டிற்கான இடைப்பயிர் சேர்க்கைகள்

| முக்கிய பயிர் | இடைப்பயிர் | கட்டுப்படுத்தும் பூச்சி[citation:3] |
|:---|:---|:---|
| முட்டைக்கோஸ் | கேரட் | Diamond Back Moth (வைர பின்புற அந்துப்பூச்சி) |
| வெண்டைக்காய் | மொச்சை | Yellow Vein Mosaic Virus (மஞ்சள் நரம்பு வைரஸ்) |
| தக்காளி | துளசி | Hornworm (கொம்பு புழு) |
| வெள்ளரி | முள்ளங்கி | வெள்ளரி வண்டு (Cucumber Beetle) |

### 5.2 மண் வளத்திற்கான இடைப்பயிர்

| இடைப்பயிர் முறை | நன்மை |
|:---|:---|
| **தோட்டக்கறி + மிளகு** (2:1 விகிதம்) | அதிக மகசூல் மற்றும் பொருளாதார வருமானம்[citation:9] |
| **தோட்டக்கறி + கடலை** (2:1 விகிதம்) | நைட்ரஜன் நிலைப்படுத்தல், மண் வளம்[citation:9] |
| **நெல் + பச்சை உரம்** | மண் வளம் மற்றும் உற்பத்தித்திறன் அதிகரிப்பு[citation:10] |

### 5.3 வாசனை செடிகள் (Aromatic Herbs) - பூச்சி விரட்டிகள்

பூச்சிகளை விரட்டுவதற்கு பின்வரும் வாசனை செடிகளை பயிர்களுக்கு இடையே அல்லது சுற்றில் பயிரிடலாம்[citation:3]:

- துளசி
- கொத்தமல்லி
- புதினா
- சாமந்தி (Marigold)
- கிரிஸான்திமம்

## 6. நோய் மற்றும் பூச்சி மேலாண்மை (IPDM - Integrated Pest & Disease Management)

### 6.1 முக்கிய பூச்சிகள் மற்றும் அவற்றின் மேலாண்மை

| பூச்சி | பாதிக்கும் பயிர் | கட்டுப்பாட்டு முறை |
|:---|:---|:---|
| **பச்சை தத்துப்பூச்சி** (Spotted Pod Borer - Maruca vitrata) | அவரை வகைகள் | பாக்கிலஸ் துரிஞ்சியென்சிஸ் (Bt) தெளிப்பு, ஃபெரமோன் பொறிகள் |
| **வெள்ளீட்டு பூச்சி** (Whitefly - Bemisia tabaci) | அவரை, தக்காளி, மிளகாய் | மஞ்சள் ஒட்டும் பொறிகள், விருந்தோம்பல் எண்ணெய் |
| **அசுவினி** (Aphid - Aphis craccivora) | அவரை, மிளகாய் | பூச்சி கவசம், லேடிபட் பூச்சிகளின் இயற்கை கட்டுப்பாடு |
| **உருளைக்கிழங்கு வெடிப்பு** (Potato Blight) | உருளைக்கிழங்கு | தாமிர அயோடைடு தெளிப்பு[citation:2] |

### 6.2 பயனுள்ள பூச்சிகள் (Beneficial Insects)

இவற்றை பாதுகாப்பது முக்கியம். IPDM முறைகள் பயனுள்ள பூச்சிகளின் எண்ணிக்கையை பாதுகாக்கின்றன[citation:12].

| பயனுள்ள பூச்சி | கட்டுப்படுத்தும் பூச்சி |
|:---|:---|
| **லேடிபட் பூச்சி** (Ladybird Beetle) | அசுவினி, செதில் பூச்சிகள் |
| **திருப்பி** (Praying Mantis) | பல்வேறு பூச்சிகள் |
| **டிரிக்கோகிரம்மா** (Trichogramma) | புழுக்களின் முட்டைகள் |
| **தேனீக்கள்** | மகரந்த சேர்க்கைக்கு முக்கியம் |
| **சிலந்திகள்** | பல்வேறு பூச்சிகள் |

## 7. பாதுகாக்கப்பட்ட சாகுபடி (Protected Cultivation) - ராசிபுரத்திற்கு சிறப்பு

ராசிபுரத்தில் மக்கள்தொகை அதிகரிப்பால் விவசாய நிலங்கள் குறைந்து வருகின்றன. பாதுகாக்கப்பட்ட சாகுபடி மிகச் சிறந்த தீர்வாகும்[citation:1].

### 7.1 பாதுகாக்கப்பட்ட சாகுபடி வகைகள்:

| வகை | அம்சம் | பொருத்தமான பயிர்கள்[citation:1] |
|:---|:---|:---|
| **பாலிஹவுஸ் / கிரீன்ஹவுஸ்** | கட்டுப்படுத்தப்பட்ட காற்றோட்டம், காப்பு | தக்காளி, குடைமிளகாய், வெள்ளரி, ஸ்ட்ராபெர்ரி, பூக்கள் |
| **ஷேட் நெட்** | அதிக சூரிய ஒளியிலிருந்து பாதுகாப்பு | நாற்றங்கால், இலை காய்கறிகள் |
| **ஹைட்ரோபோனிக்ஸ்** | மண் இல்லாமல், 90% நீர் சேமிப்பு | கீரை, தக்காளி, மிளகாய் |

### 7.2 பாதுகாக்கப்பட்ட சாகுபடியின் நன்மைகள்[citation:1]:

1. **அதிக மகசூல்**: திறந்த வயலை விட 5 மடங்கு வரை
2. **ஆண்டு முழுவதும் சாகுபடி**: நிலையான வருமானம்
3. **பாதுகாப்பு**: பூச்சிகள், நோய்கள், கடுமையான வானிலையிலிருந்து
4. **நீர் சேமிப்பு**: மேம்பட்ட பாசன அமைப்புகள்

## 8. குறிப்பிட்ட ஊட்டச்சத்து தேவைகள் அடிப்படையில் பயிர்கள்

| ஊட்டச்சத்து | அதிகம் உள்ள பயிர்கள்[citation:3] |
|:---|:---|
| **புரதம்** | மொச்சை, கொத்தவரை, பிரெஞ்ச் பீன்ஸ், அவரை |
| **மாவுச்சத்து** | வேர்கள் & கிழங்குகள், வாழை |
| **கால்சியம்** | பீட், அரைக்கீரை, வெந்தயம், முள்ளங்கி, கொத்தமல்லி, பூசணி, வெங்காயம், தக்காளி |
| **பொட்டாசியம்** | சர்க்கரை வள்ளிக்கிழங்கு, உருளைக்கிழங்கு, பாகற்காய் |
| **பாஸ்பரஸ்** | பூண்டு, பட்டாணி, பாகற்காய் |
| **இரும்பு** | பாகற்காய், முருங்கை, அரைக்கீரை, வெந்தயம், கீரை |
| **வைட்டமின் A** | கேரட், ப்ரோக்கோலி, பூசணி, சர்க்கரை வள்ளிக்கிழங்கு, முட்டைக்கோஸ், தக்காளி, கொத்தமல்லி |
| **வைட்டமின் C** | மிளகாய், ப்ரோக்கோலி, உருளைக்கிழங்கு, எலுமிச்சை, முள்ளங்கி, காலிஃப்ளவர், பாகற்காய், முள்ளங்கி |
| **வைட்டமின் K** | இலை காய்கறிகள், ப்ரோக்கோலி, காலிஃப்ளவர், முட்டைக்கோஸ் |

## 9. ராசிபுரத்தில் பரிந்துரைக்கப்படும் பயிர் சுழற்சி மாதிரி

கீழே உள்ள 4-வருட சுழற்சி மாதிரியைப் பின்பற்றலாம்[citation:3]:

| வருடம் | பருவம் | முக்கிய பயிர் | இடைப்பயிர் |
|:---|:---|:---|:---|
| **1** | காரிஃப் | நெல் | - |
| **1** | ராபி | உருளைக்கிழங்கு / கேரட் | முள்ளங்கி |
| **2** | காரிஃப் | வெண்டைக்காய் | மொச்சை |
| **2** | ராபி | முட்டைக்கோஸ் / காலிஃப்ளவர் | கேரட் |
| **3** | காரிஃப் | தக்காளி | துளசி |
| **3** | ராபி | பட்டாணி / அவரை | - |
| **4** | காரிஃப் | பச்சை உரம் (மொச்சை/கொத்தவரை) | - |
| **4** | ராபி | கத்திரிக்காய் / மிளகாய் | வெந்தயம் |

## 10. வேகமாக அறுவடை செய்யும் காய்கறிகள்

விரைவான மகசூலுக்கு கீழே உள்ள பயிர்களை தேர்வு செய்யவும்[citation:3]:

| பயிர் | அறுவடை காலம் |
|:---|:---|
| கீரை வகைகள் | 20-30 நாட்கள் |
| முள்ளங்கி | 25-35 நாட்கள் |
| வெண்டைக்காய் | 45-50 நாட்கள் |
| பீர்க்கங்காய் | 50-60 நாட்கள் |
| கத்திரிக்காய் | 60-70 நாட்கள் |
| தக்காளி | 70-80 நாட்கள் |
